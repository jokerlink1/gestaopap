# gestaopap (Release-Only Repository)

Este repositório publica apenas artefactos compilados da aplicação.

## Política de publicação

- Não contém ficheiros fonte `.py`.
- Não contém código não compilado.
- Apenas ficheiros compilados para distribuição ao utilizador final.

## Conteúdo atual

- `RELEASES/FINAL_V60.pyc`
- `RELEASES/servidor_juri.pyc`

## Atualizações

Sempre que existir nova versão:

1. Compilar os ficheiros necessários.
2. Atualizar os binários em `RELEASES/`.
3. Atualizar `update_manifest.json`.
4. Fazer commit e push.
